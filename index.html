    QC Reports

    • Sequence source attributes that have been curator-edited in MGI and whose incoming (new) value is not equal to its existing MGI value
    • Organisms that could not be translated to a valid MGI Organism
    • Cell Lines that could not be translated to a valid MGI Cell Line
    • Genders that could not be translated to a valid MGI Gender
    • Libraries that could not be translated to a valid MGI Library
    • Strains that could not be translated to a valid MGI Strain
    • Tissues that could not be translated to a valid MGI Tissue
    • Discrepencies between the Clone Libraries of any of a Sequence's associated Clones
    • Sequences that are designated as Merged by the Sequence provider
    • Sequence/Reference associations that are no longer in the Sequence provider record
    • Sequence Raw Source Attributes (Library or Organism only) where its new raw value differs from its old raw value
